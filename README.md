# 🏪 لوحة إدارة المتجر — PWA v2

تطبيق ويب قابل للتثبيت (PWA) لإدارة متجرك الإلكتروني بشكل احترافي وسهل.

## ✨ المميزات

- 📦 إدارة المنتجات (إضافة، تعديل، حذف)
- 🖼️ رفع حتى 5 صور لكل منتج مع ضغط تلقائي
- 🎨 دعم متغيرات المنتج (أحجام، ألوان، ...)
- 🔥 قسم العروض منفصل عن القسم الرئيسي
- ⠿ إعادة ترتيب المنتجات بالسحب والإفلات
- 🔍 بحث وتصفية سريع
- 📈 خطة نمو أسبوعية مدمجة (4 أسابيع)
- 📱 PWA قابل للتثبيت على الجوال
- 🌙 وضع مظلم بالكامل

## 🚀 خطوات الرفع على GitHub

### ١. إنشاء مستودع جديد
1. اذهب إلى [github.com](https://github.com) وسجّل دخولك
2. اضغط **New repository**
3. اختر اسماً مثل `store-dashboard`
4. اجعله **Public**
5. اضغط **Create repository**

### ٢. رفع الملفات
ارفع هذه الملفات على المستودع:
```
index.html
manifest.json
sw.js
icons/
  icon-192.png
  icon-512.png
.github/workflows/deploy.yml
```

### ٣. تفعيل GitHub Pages
1. اذهب إلى **Settings** → **Pages**
2. Source: **GitHub Actions**
3. انتظر دقيقتين
4. الرابط: `https://USERNAME.github.io/REPO-NAME/`

### ٤. إنشاء مفتاح الربط
في متصفحك، افتح Console وأدخل:
```js
btoa(unescape(encodeURIComponent(JSON.stringify({
  u: "اسم_المستخدم",
  r: "اسم_المستودع",
  t: "github_token"
}))))
```

### ٥. تثبيت التطبيق
- افتح الرابط من **Chrome** على الجوال
- ستظهر رسالة "تثبيت التطبيق" — اضغط عليها
